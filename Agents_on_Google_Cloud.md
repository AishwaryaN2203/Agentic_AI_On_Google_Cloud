# Agent
- Observe the world
- Interact with the world with the tools provided to it
- Achieves complex, specific goals
- Agents can take action on our behalf, or even act autonomously to accomplish a goal, or perhaps find a new goal

- Chatbots takes a set of pre-determined actions to reach a goal, **Agents** take autonomous actions, enabling them to set goals and carry them out independently

### Core capabilities
Agents have various features that work together to understand the goals, plan steps, and interactions with other systems to achieve the goal.

1. Autonomous Actions: Agents can perform complex tasks and actions with minimal human interactions
2. Reasoning and Planning: Agents make informed decisions and adapt to changing environments 
3. Continuous Learning: Agents can learn from experience and improve themselves over time

**Multi Agent Orchestration**
- Allows agents to share information, plan/ divide, and coordinate tasks to achieve the large-scale goals that a single agent cannot accomplish.
